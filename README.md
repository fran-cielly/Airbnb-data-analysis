## Airbnb Exploratory Data Analysis with Python
![EN-US](https://img.shields.io/badge/EN--US-003366?style=flat&logo=united-states&logoColor=white)
This project performs a Exploratory Data Analysis on the Airbnb housing market. <br>The main objective is to apply Data Science techniques to clean, process, and extract intelligence from raw data.
The development focused on using the Python ecosystem for data manipulation and visualization, identifying pricing patterns, geographical distribution, and correlations between variables that impact consumer choice.

<hr style="height: 1px;/>

![PT-BR](https://img.shields.io/badge/PT--BR-009c3b?style=flat&logo=brazil&logoColor=white)
Este projeto consiste em uma análise exploratória de dados sobre o mercado de hospedagens do Airbnb.<br> O objetivo principal é aplicar técnicas de Ciência de Dados para limpar, processar e extrair inteligência de dados brutos.
O foco do desenvolvimento foi a utilização do ecossistema Python para manipulação e visualização de dados, identificando padrões de preços, distribuição geográfica e correlações entre variáveis que impactam a escolha do consumidor.
