This project consists of an Exploratory Data Analysis (EDA) of the Airbnb rental market. 
The main objective is to apply data techniques to clean, process, and extract insights from raw data. I used Python for data manipulation and visualization.
<ul>
  <li>Project Scope: Exploratory Data Analysis (EDA) of the Airbnb rental market</li>
  <li>Objective: Apply data techniques to clean, process, and extract insights from raw data</li>
  <li>Tools: Python was used for data manipulation and visualization</li>
</ul>

### Exploratory Analysis
<ul>
  <li>Which are the top 5 neighborhoods with the most listings?</li>
  <li>What is the proportion of property types?</li>
  <li>What is the average price by neighborhood?</li>
  <li>What is the property availability?</li>
</ul>

Este projeto consiste em uma análise exploratória de dados sobre o mercado de hospedagens do Airbnb.
O objetivo principal é aplicar técnicas de dados para limpar, processar e extrair informações de dados brutos. Utilizei Python para manipulação e visualização de dados.
<ul>
  <li>Escopo do Projeto: Análise Exploratória de Dados (EDA) do mercado de hospedagens do Airbnb</li>
  <li>Objetivo: Aplicar técnicas de dados para limpar, processar e extrair insights de dados brutos</li>
  <li>Ferramentas: Utilização de Python para manipulação e visualização de dados</li>
</ul>

### Análise exploratória
<ul>
  <li>Quais são os 5 bairros com maior número de anúncios?</li>
  <li>Qual a proporção entre os tipos de imóveis?</li>
  <li>Qual o preço médio nos bairros?</li>
  <li>Qual a disponibilidade dos imóveis?</li>
</ul>
